# 📦 КОРОБОЧКА 5.0 — Анализ проекта

## 📖 Описание

Мотивационное приложение по методике психолога Виолетты Макеевой. Помогает организовать задачи по категориям и выполнять их через случайный выбор с таймером фокусировки. Работает полностью офлайн как PWA.

---

## 🏗️ Архитектура

### Стек технологий
- **Frontend:** React 19 + TypeScript + Vite
- **Стилизация:** Tailwind CSS v4 с семантическими токенами (HSL)
- **Хранение данных:** IndexedDB (библиотека — собственная обёртка)
- **PWA:** vite-plugin-pwa + Service Worker
- **Уведомления:** Web Notifications API

## 🗂️ Система категорий

| ID | Название | Цвет | Назначение |
|----|----------|-------|------------|
| 0  | Не определена | Серый | Задачи без категории |
| 1  | Обязательные | Жёлтый | Срочные и важные дела |
| 2  | Безопасность | Синий | Базовые потребности |
| 3  | Простые радости | Зелёный | Приятные мелочи |
| 4  | Эго-радости | Красный | Статус и признание |
| 5  | Доступность простых радостей | Голубой | Условия для радостей |

Секции дашборда группируют категории:
- **Обязательные дела** → [1]
- **Система безопасности и доступность** → [2, 5]
- **Простые и эго-радости** → [3, 4]

## ⚙️ Функциональные модули

### 1. Управление задачами (`Task`)
- **Поля:** id, text, category, completed, active, statusChangedAt, subcategory, timeSpent, templateId
- Добавление (по одной или пакетно через `\n`)
- Редактирование inline
- Переключение active/completed
- Подкатегории (предустановленные + пользовательские)

### 2. Случайный выбор задачи
- Из активных незавершённых задач выбранной секции
- Запускает полноэкранный таймер с фоновым изображением категории

### 3. Таймер (`TimerScreen`)
- Настраиваемая длительность (по умолчанию 10 мин)
- Обратный отсчёт с отображением прошедшего времени
- Трекинг `timeSpent` — накопительный учёт секунд
- По завершении: «Завершить» или «Вернуть» задачу
- Фоновое изображение из `Assets/` по категории

### 4. Повторяющиеся шаблоны (`TaskTemplate`)
- Типы: ежедневно / еженедельно / ежемесячно
- Настройка: день недели, день месяца, час создания
- Авто-создание задач при загрузке и каждые 30 мин
- Хранение в IndexedDB (ключ `templates` в meta-store)

### 5. Статистика по времени (`TimeStatsPanel`)
- Общее время по категориям
- Топ задач по потраченному времени
- Визуализация прогресса

### 6. Импорт / Экспорт
- Экспорт всех задач в JSON-файл с датой в имени
- Импорт из JSON-файла с валидацией

### 7. Онбординг (`OnboardingScreen`)
- Показывается один раз при пустом списке задач
- Пошаговое знакомство с функциями
- Флаг `onboarding_done` в localStorage

### 8. Тема оформления
- Светлая / тёмная тема через CSS-переменные
- Переключатель в хедере
- Сохранение выбора в localStorage

### 9. Push-уведомления
- Запрос разрешения через `NotificationButton`
- Напоминания через Web Notifications API

## 💾 Хранение данных

### IndexedDB (база `korobochka-db`)
- **Store `tasks`:** все задачи (keyPath: `id`)
- **Store `meta`:** произвольные данные (templates, customSubcategories)

### Миграция
- Однократная миграция из localStorage в IndexedDB
- Флаг `idb_migrated` в localStorage

### Кэширование
- `cachedTasks` — in-memory кэш для синхронного доступа
- Асинхронная запись в IndexedDB при каждом изменении

## 🔄 Поток данных

```
App.tsx (AppContext)
  ├── tasks: Task[]           ← IndexedDB → cachedTasks
  ├── setTasks(fn)            → обновляет state + сохраняет в IDB
  ├── openTimer(task)         → показывает TimerScreen
  └── openAddModal(cat?)      → показывает AddTaskModal

Dashboard → считает active/completed → кнопки 🎲 и ➕
TaskListPanel → группировка по категориям → inline-редактирование
TimerScreen → обратный отсчёт → timeSpent tracking → complete/return
TemplatesPanel → CRUD шаблонов → auto-create задач
TimeStatsPanel → агрегация timeSpent по категориям
```
